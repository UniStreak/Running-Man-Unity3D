# Running-Man Unity Game

There is a wood man that is our main character. Wood man tries to go as far as he can go. He needs to use bonus barrels for maximum distance.

Manual:  Press space to play. In game, press space to jump and press mouse button to use boosters. If score reaches "100", the upper platforms appears. Friction is disabled on upper platforms and these platforms are shorter. If you fall down from the platforms, game-over.
